# Flask Task Manager

O aplicație web pentru gestionarea sarcinilor, construită cu Python și Flask.

## Detalii Tehnice
* **Backend:** Flask & SQLAlchemy.
* **Bază de date:** SQLite (stocare persistentă).
* **Frontend:** Bootstrap 5.

## Funcționalități
* Adăugarea de task-uri cu termen personalizat.
* Ștergerea task-urilor finalizate.
